## List methods

- slicing
    - a[0:2] == a[:2]
    - a[2:] == a[2:a.len()]

- plus
    - a + b
    - a * 3
    - a.extend(b)

- modify
    - a[2] = 4
    - a[1:2] = ['a','b','c'] : insert
    - a[1] = ['a','b','c'] : replace
    - del a[x:y] / del a[x]

- append
- sort
    - a.sort()
    - a.reverse()
    
- a.index(number)
    - List doesn't have `find` method

- insert
    - a.insert(0,4) insert 4 in 0 index

- remove
    - a.remove(x)    

- pop
- count
